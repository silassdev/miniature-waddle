# ShepherdAI

## Overview
ShepherdAI is a web application designed to provide compassionate, scripture-rooted guidance and spiritual support through advanced artificial intelligence. Built with Next.js, it combines modern web technology with biblical wisdom to assist users in their daily spiritual walk.

## Features
- AI Spiritual Assistant: Conversational interface powered by Google Gemini, providing prayerful reflections and scriptural insights.
- Daily Bread: curated spiritual content for daily consumption.
- Prayer Wall: A community-focused space for sharing and supporting prayer requests.
- Secure Authentication: Multi-factor and OAuth-based security using NextAuth.js.
- Responsive Design: A premium, adaptive user interface that works seamlessly across desktop and mobile devices.

## Technology Stack
- Framework: Next.js 15
- Language: TypeScript
- Database: MongoDB via Mongoose
- Authentication: NextAuth.js
- AI Integration: Google Generative AI (Gemini)
- Styling: Vanilla CSS with custom design tokens
- Animations: Framer Motion
- Icons: React Icons (Feather)

## Getting Started

### Prerequisites
- Node.js 18.x or higher
- MongoDB instance (local or Atlas)
- Google Cloud Project with Gemini API access



## Project Structure
- /app: Next.js App Router pages and API routes.
- /app/components: Shared UI components.
- /lib: Utility functions and database configuration.
- /models: Mongoose database schemas.
- /public: Static assets and images.
- /styles: Global CSS and design system.
